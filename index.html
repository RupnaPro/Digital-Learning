Ans: The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. The core characteristic of a semantic element is that it clearly communicated its meaning to both the developer and the browser. These elements clearly define its content. For example -< header> -defines a header for a web page, <nav> -defines a container for navigation links, <section> -defines a section in a web page, <article> -element contains the main part, containing information about the web page, <aside> -content is often placed as a sidebar in a document, <footer> -defines a footer for a document or a section.
